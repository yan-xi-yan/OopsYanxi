# 一级标题示例

段落之间使用空行分隔。

这是第二段。_斜体_、**粗体** 和 `等宽文本` 都可以正常使用。无序列表看起来像这样：

- 这一项
- 那一项
- 另一项

注意，先不考虑项目符号，实际文本内容是从缩进 4 个空格的位置开始的。

> 引用块就是这样写的。
>
> 如果你愿意，
> 它也可以跨越多个段落。

使用 3 个连字符表示破折号。使用 2 个连字符表示范围，例如“第 12--14 章”。三个点 `...` 会被转换成省略号。也支持 Unicode，例如：你好，世界。

## 二级标题示例

下面是一个有序列表：

1. 第一项
2. 第二项
3. 第三项

再次注意，实际文本从左侧第 4 个字符的位置开始。下面是一个代码示例：

    # 我再重复一遍……
    for i in 1 .. 10 { do-something(i) }

正如你可能已经猜到的那样，这是通过缩进 4 个空格实现的。顺便说一下，你也可以不用缩进，而是使用定界代码块：

```
define foobar() {
    print "欢迎来到风味之国！";
}
```

（这样复制粘贴会更方便。）你还可以为定界代码块标注语言，让 Pandoc 进行语法高亮：

```python
import time
# 快，数到十！
for i in range(10):
    # （但也别太快）
    time.sleep(0.5)
    print(i)
```

### 三级标题示例

现在来看一个嵌套列表示例：

1. 先准备这些食材：

    - 胡萝卜
    - 芹菜
    - 扁豆

2. 烧一锅水。

3. 把所有东西倒进锅里，然后按这个流程操作：

        找到木勺
        打开锅盖
        搅拌
        盖上锅盖
        把木勺摇摇晃晃地架在锅柄上
        等 10 分钟
        回到第一步（或者煮好后关火）

    不要碰到木勺，不然它会掉下来。

再次注意，文本总是对齐在 4 个空格的缩进位置上，包括上面第 3 项后面继续的那一行也是如此。

这里有一个指向 [网站](http://foo.bar) 的链接，一个指向 [本地文档](local-doc.html) 的链接，以及一个指向 [当前文档某个标题](#二级标题示例) 的链接。这里还有一个脚注 [^1]。

[^1]: 脚注内容写在这里。

表格也可以写成这样：

Table: 鞋子、尺码以及材质

（上面这一行是表格标题。）
