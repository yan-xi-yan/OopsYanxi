# 多元线性回归算法

## 1. 从一元到多元

> **类比**：预测房价时，不只看面积（一元），还要考虑楼层、地段、房龄等多个因素——这就是多元线性回归。

**模型形式**：

$$\hat{y} = w_1x_1 + w_2x_2 + \cdots + w_nx_n + b$$

向量形式：$\hat{y} = \mathbf{w}^T\mathbf{x} + b$

---

## 2. 损失函数

使用**均方误差 (MSE)** 作为损失：

$$L(\mathbf{w}, b) = \frac{1}{m}\sum_{i=1}^{m}(\hat{y}^{(i)} - y^{(i)})^2$$

目标：找到使 $L$ 最小的 $\mathbf{w}$ 和 $b$。

---

## 3. 求解方法

### 3.1 正规方程（解析解）

$$\mathbf{w} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$$

- **优点**：一步到位，无需迭代
- **缺点**：特征数量大时矩阵求逆计算量为 $O(n^3)$，不适合高维数据

### 3.2 梯度下降（迭代解）

见 [梯度下降法与模型训练](../03_Model_Training/01_梯度下降法与模型训练.md)

---

## 4. Python 示例

```python
from sklearn.linear_model import LinearRegression
import numpy as np

# 训练数据：[面积, 楼层, 房龄]
X = np.array([[80, 5, 3], [120, 10, 8], [60, 2, 1]])
y = np.array([300, 500, 200])  # 房价（万元）

model = LinearRegression()
model.fit(X, y)

print("系数:", model.coef_)       # 每个特征的权重
print("截距:", model.intercept_)  # 偏置项
print("预测:", model.predict([[100, 7, 5]]))
```

---

## 5. 注意事项

> **多重共线性**：若特征之间高度相关（如"面积"和"房间数"），会导致 $\mathbf{X}^T\mathbf{X}$ 接近奇异，权重估计不稳定。解决方案：使用岭回归（L2正则化）。

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| 多重共线性 | 权重方差大、符号异常 | 岭回归、删除冗余特征 |
| 特征量纲不一 | 梯度下降收敛慢 | 归一化/标准化 |
| 非线性关系 | 残差有规律 | 升维（多项式特征） |
