# 04 - Agent 与工具使用

> 前面几章中，LLM 只是"被动回答"——你问它答。Agent 让 LLM 学会**主动行动**：观察环境、选择工具、执行操作、根据结果继续推理，直到完成任务。

## 本章笔记

1. [[01_工具系统]] — Tool 定义、`@tool` 装饰器、StructuredTool、内置工具与异步工具
2. [[02_Agent架构与实战]] — ReAct 推理循环、Tool Calling Agent、AgentExecutor 与多工具实战

## 学完本章你将掌握

- 理解 Agent 与普通 Chain 的本质区别
- 使用 `@tool` 装饰器定义自定义工具
- 掌握 ReAct 推理循环（Reasoning + Acting）的工作原理
- 构建一个能调用搜索、计算等外部工具的自主 Agent
