# 06 - LangGraph 入门

> AgentExecutor 适合简单场景，但面对多步推理、条件分支、人机协作时力不从心。LangGraph 用**有向图**来定义 Agent 逻辑，是 LangChain 官方推荐的下一代 Agent 框架。

## 本章笔记

1. [[01_LangGraph入门]] — StateGraph 核心概念、节点与边、条件路由、人机协作、持久化

## 学完本章你将掌握

- 理解 LangGraph 与 AgentExecutor 的区别与适用场景
- 使用 StateGraph 定义节点（Node）和边（Edge）
- 实现条件分支与循环推理
- 加入 Human-in-the-loop 人工审批节点
- 使用 Checkpointer 实现状态持久化与断点恢复
